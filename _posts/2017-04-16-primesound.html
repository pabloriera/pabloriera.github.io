Are prime number distributed randomly?...... Nope.

One way to see prime numbers is as special cases of the divisor function $\sigma_0(n)$, the function that counts how many divisor a number $n$ has. A number is prime if the function equals two. The following image contains first 20000 divisor function values for natural numbers from top left to bottom right. The value two is highlighted.
